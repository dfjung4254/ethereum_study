## Solidity 개발환경 구축
```
1. Remix(Solidity IDE on Cloud) : https://remix.ethereum.org/
2. solcjs(compiler)
  $ npm install -g solc
  $ solcjs  //실행
3. Docker build
  $ docker run ethereum/solc:stable --version

```
